Demo v6 project
===============

This is the demo v6 project packaging.

Clone the modules
-----------------

Only needed once:

```bash
./clone.sh
```

Pull the latest modules
-----------------------

Only needed if the modules were already cloned: 

```bash
./pull.sh
```

Build the image with the modules
--------------------------------

```bash
docker compose build
```

Run
---

```bash
docker compose up
```
